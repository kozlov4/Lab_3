# 📱 Каталог Курсів — Android App (Lab 3)

Мобільний застосунок для платформи Android, розроблений мовою **Kotlin** в рамках лабораторної роботи №3 з дисципліни «Високорівневі мови програмування та фреймворки».

Є мобільним аналогом веб-застосунку з лабораторної роботи №2 (Node.js + React).

---

## 🛠 Технології

- **Kotlin** — мова розробки
- **Android Studio** — середовище розробки
- **RecyclerView** — відображення динамічних списків
- **AlertDialog** — діалоги взаємодії з користувачем
- **LinearLayout / XML** — розмітка інтерфейсу

---

## ✅ Реалізований функціонал

### Рівень 1 — Каталог курсів
- Перегляд списку доступних курсів
- Відображення статусу кожного курсу: `Не придбано` / `В процесі` / `Завершено`
- Вкладка «Мої курси» з фільтрацією придбаних

### Рівень 2 — Запис та відгуки
- Запис на курс (зміна статусу на «В процесі»)
- Завершення курсу
- Залишення відгуку з оцінкою від 1 до 5
- Динамічний перерахунок середнього рейтингу

### Рівень 3 — Авторизація та програми
- Авторизація користувача при запуску (діалог введення логіну)
- Перегляд навчальних програм («Fullstack Розробник», «Backend Майстер»)
- Відстеження прогресу: скільки курсів програми пройдено

---

## 🚀 Запуск

1. Відкрити проект у **Android Studio**
2. Запустити на емуляторі або фізичному пристрої через `Run > Run 'app'`
3. Мінімальна версія Android: **API 24 (Android 7.0)**

---

```
app/src/main/
```
